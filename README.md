## Run the application with python
#### Install poetry
```shell
pip install poetry
```
 
#### Install the project dependencies
```shell
cd src && poetry install
```

#### Spawn a shell within the virtual environment
```shell
poetry shell
```

#### Run the application
```shell
python3 main.py
```
